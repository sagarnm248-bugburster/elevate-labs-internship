# 📊 Day-5: Exploratory Data Analysis (EDA)

## 📌 Objective
Perform exploratory data analysis to identify patterns, trends, and relationships in the dataset.

---

## 🛠 Tools Used
- Python (Pandas, Matplotlib, Seaborn)

---

## 📂 Files Included
- eda.ipynb → Analysis Notebook  
- report.pdf → Summary of findings  

---

## 📊 Analysis Performed
- Data inspection using .info() and .describe()
- Missing value analysis and handling
- Univariate analysis (distribution plots)
- Bivariate analysis (comparisons)
- Correlation heatmap

---

## 📈 Key Insights
- Female passengers had higher survival rate
- First-class passengers survived more
- Age group 20–40 dominates dataset
- Fare and class show correlation

---

## ✅ Outcome
Gained insights into data patterns and improved data analysis and visualization skills.

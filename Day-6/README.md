# 📊 Day-6: Sales Trend Analysis Using Aggregations

## 📌 Objective
Analyze monthly revenue and order volume using SQL aggregation functions.

## 🛠 Tools Used
- MySQL / PostgreSQL

## 📂 Files Included
- task6.sql → SQL queries
- output.png → Results screenshot

## 📊 Analysis Performed
- Extracted year and month from order_date
- Calculated total revenue using SUM()
- Calculated order volume using COUNT(DISTINCT)
- Grouped and sorted results

## 📈 Key Insights
- Monthly revenue trends identified
- Order volume varies across months
- Peak sales months identified

## ✅ Outcome
Learned how to group data and analyze time-based trends using SQL.

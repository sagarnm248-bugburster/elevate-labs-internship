# 📊 Task 2 - Data Visualization and Storytelling

## 🎯 Objective

To create a Power BI dashboard that provides meaningful insights from sales data and demonstrates data storytelling.

## 🛠 Tools Used

* Power BI

## 📁 Dataset

* Superstore Sales Dataset

## 📊 Visualizations Created

* Sales by Category
* Sales Trend Over Time
* Sales by Region
* Profit by Category

## 🔍 Key Insights

* Technology category generates the highest sales
* Sales show a decreasing trend over time
* West region contributes the most revenue
* Furniture category has lower profit compared to others

## 📌 Outcome

Gained hands-on experience in creating interactive dashboards and deriving business insights through data visualization.
